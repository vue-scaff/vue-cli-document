# 开始使用 Vue-Scaff

# 安装
yarn global add @vue/cli
# 或者
npm i -g @vue/cli

# 创建一个应用
vue create -p vue-scaff/vue-cli-preset --clone <project-name>

# 开始编码
yarn serve

# 构建项目
yarn build

注意

请确保您的 Node.js 版本 >= 10。
