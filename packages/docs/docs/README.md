### Get started with Vue-Scaff

```sh
# install
yarn global add @vue/cli
# or
npm i -g @vue/cli

# create a application
vue create -p vue-scaff/vue-cli-preset --clone <project-name>

# start coding
yarn serve

# build project
yarn build
```

::: warning COMPATIBILITY NOTE
Vue-Scaff requires Node.js >= 10
:::
