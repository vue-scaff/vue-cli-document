Check out docs at http://vue-scaff.joenix.com/.

## Get Started

```sh
# install basic dependencies
yarn global add @vue/cli

# create an application
vue create -p vue-scaff/vue-cli-preset --clone <project-name>

# start coding
yarn serve

# build project
yarn build
```
